Welcome to the Toggle Slider Tutorial

By Ben Abroms

In this tutorial, we will learn how to make a toggleable slider that can switch from one side and display that it is on.

The Toggleable Slider above is the one we will be creating for this tutorial.

First, we will need to make a label with a class called "switch.

<label class="switch">
<input type="checkbox">
10%</div>
<span class="slider round"></span></label>

We made a input type that's a checkbox and made a span class called "slider". Sliders are more noticeable than a basic checkbox.

Next, we'll need to make a style in CSS for the design of the Slider itself and the way it works.

.switch {
position: relative;
display: inline-block;
width: 60px;
height: 34px;
}
.switch input {display:none;}

We set up two CSS effects. The first one is responsible for making the checkbox into a switch and the second one hides the default checkbox.

Now we need to make the slider "animate" by applying certain CSS properties.

.slider {
position: absolute;
cursor: pointer;
top: 0;
left: 0;
right: 0;
bottom: 0;
background-color: #ccc;
-webkit-transition: .4s;
transition: .4s;
}
.slider:before {
position: absolute;
content: "";
height: 26px;
width: 26px;
left: 4px;
bottom: 4px;
background-color: white;
-webkit-transition: .4s;
transition: .4s;
}
input:checked + .slider {
background-color: #2196F3;
}
input:focus + .slider {
box-shadow: 0 0 1px #2196F3;
}
input:checked + .slider:before {
-webkit-transform: translateX(26px);
-ms-transform: translateX(26px);
transform: translateX(26px);
}
.slider.round {
border-radius: 34px;
}
.slider.round:before {
border-radius: 50%;
}

This all looks complicated, but it's not. First, we had to design the slider itself such as where it is on the screen, the size, color, and we made a webkit transition property of .4 seconds. That controls the amount of time it takes to switch to the on side. Second, we made show what the slider will be in its default state or off. Next, we use the "input:checked + .slider to change the background color after the toggle. This is the same case with the "input:focus + .slider except that controls the box shadow. The The "input:checked + .slider:before allows the circle in the middle to move into the new position after it has been toggled. The last two are styled options that you don't have to do unless you want one with a rounded border like the example at the top.

Here is the final result! Your brand new Toggleable Slider!
